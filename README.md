# Snake Game

A classic Snake game built with HTML, CSS, and JavaScript.
You control the snake, eat food to grow, and try not to hit the walls or your own tail.
The game runs directly in the browser and works smoothly on desktop and mobile.

## Features

- Smooth snake movement
- Score tracking
- Food spawns at random positions
- Game over detection
- Clean layout
- Mobile-friendly controls
- Mobile-friendly controls

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**

## How to Run

Download or clone the project:

```bash
git clone https://github.com/jvkedev/snake-game.git
```

Open index.html in your browser.

## Notes

- This is a static frontend project.
- No backend, database, authentication or API calls are used.
- Perfect for beginners practicing DOM manipulation and game logic.
